1. Ключевые AI-навыки и практические кейсы

1.1 Эффективная работа с нейросетями

  • Различаю ИИ, нейросети, ChatGPT, умею объяснять разницу простым языком.
  • Экономлю время на поиске информации: чат-боты вместо Google.
  • Применяю нейросети для личной эффективности: составление диет, подбор тренировок, обучение языкам.
📌 Пример кейса: Оптимизировала рутинные задачи в финансово-экономическом отделе, автоматизировав поиск необходимой финансовой информации для оперативных отчетов.

1.2 Промт-инжиниринг

  • Владею всеми техниками: последовательные, ролевые, диалоговые промты, кастомизация под задачу.
  • Использую фреймворки для генерации сложных бизнес-текстов и аналитики.
📌 Пример кейса: Разработала промты для генерации управленческих решений в сложных ситуациях - получила сокращение времени на анализ на 70%.

1.3 Работа с текстами и контентом

  • Автоматизация написания регламентов, писем, презентаций, аналитических записок.
  • Использую нейросети для генерации контент-стратегий, статей, презентаций, КП.
📌 Пример кейса: С помощью ChatGPT и Midjourney создала более 30 презентаций и статей для отдела маркетинга.

1.4 Аналитика и финансы

  • Автоматизирую обработку выписок, подготовку финансовых моделей, анализ отчетности.
  • Разработка дашбордов и методик финансового анализа с помощью AI.
📌 Пример кейса: Построила систему автоматического анализа договоров и выявления рисков - 90% кейсов закрываются без юриста.

1.5 Автоматизация процессов

  • Интеграция нейросетей с Google Таблицами, автоматизация расчетов, нормализация данных.
  • Внедрение AI-ассистентов в бизнес-процессы (финансы, HR, продажи).
📌 Пример кейса: Написала скрипт для автосоздания KPI-дашбордов в Google Sheets с использованием GPT.

1.6 Уникальные решения для бизнеса

  • Автоматизация согласования заявок через Telegram-ботов.
  • Построение системы "цифрового ассистента для руководителя".
📌 Пример кейса: Реализовала систему моментальных бонусов через Telegram-бота и Google Таблицу (case inside).
